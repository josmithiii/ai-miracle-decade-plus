# ai-miracle-decade-plus

Landing page for the **AI Miracle Decade Plus** course — a self-paced reading
of every landmark AI paper from DBN (2006) to GPT-4 (2023).

- **Live page:** <https://josmithiii.github.io/ai-miracle-decade-plus/>
- **Course content:** <https://github.com/josmithiii/Courses/tree/main/ai-miracle-decade-plus>
- **Companion paper repo:** <https://cm-gitlab.stanford.edu/jos/music423-2023>

The site is a single self-contained `index.html` — no build step, no
dependencies. Edit and push; GitHub Pages serves it from the `main` branch.
